Privacy Policy

Effective date: April 15, 2026

CCF DDL Tracker does not require an account and does not intentionally collect your personal data.

Local Storage

Deadlines, imported conference entries that you save, and interface preferences are stored in chrome.storage.local within the browser profile where the extension runs.

External Data Sources

The extension can fetch conference information from the upstream CCFDDL repository and fallback ICS feeds. These sources are used to help populate the import list.

Analytics

The extension workflow is designed to operate without project-hosted telemetry, user accounts, or cloud sync managed by this project.

Third-Party Links

The extension and this website may link to third-party sites such as conference homepages, GitHub repositories, or the Chrome Web Store. Those services have their own privacy policies and terms.

Changes

If this policy changes, the updated version will be published on this page.

Contact

For questions about the project, open an issue on GitHub.
