Guide

This page covers the basic workflow for using CCF DDL Tracker after installation.

Open the Popup

Pin the extension to the Chrome toolbar and click the icon. The extension opens as a native popup, so you can check deadlines without leaving the current tab.

Add a Deadline Manually

Use the add form to enter a title, date, and time. Saved deadlines are displayed in time order, and the nearest item drives the toolbar badge countdown.

Import Conferences from CCFDDL

The import panel can fetch recommended conferences from the upstream CCFDDL data source. Imported cards keep their official homepage links, and you can add selected items into your own deadline list.

Tip

Imported deadlines are helpful for discovery, but you should still confirm the final submission schedule on the official conference website.

Customize the Display

  • Switch language between Chinese and English from the bottom toolbar.
  • Choose a 24-hour or 12-hour clock in settings.
  • Change date order between year-first and month-first formats.

Open Official Conference Sites

Imported conferences retain their homepage links, so once a card is added to your list, you can jump from the popup to the official website for calls for papers or updates.

Storage Model

All deadlines and preferences are stored in chrome.storage.local. The extension does not require sign-in or cloud sync to function.
