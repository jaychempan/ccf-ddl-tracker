CCF DDL Tracker

An open source Chrome extension for tracking CCF deadlines.

Get Started GitHub
Version 2.0

Compact popup workflow for dense conference schedules

The popup focuses on quick scanning: compact top cards, a persistent import panel, a floating conference picker, footer shortcuts, and lightweight settings.

CCF DDL Tracker provides a compact popup for keeping deadlines visible in Chrome. You can add custom deadlines, import recommended conferences from CCFDDL, and keep everything stored locally in your browser.

Features

  • Focused tracking
    • Keep deadlines in a dense popup instead of a separate dashboard.
    • See remaining days quickly with compact cards and sorted lists.
  • CCFDDL import flow
    • Import recommended conferences from the CCFDDL dataset and ICS fallback.
    • Keep homepage links so each card can jump to the official conference site.
  • User-friendly settings
    • Switch between Chinese and English from the bottom toolbar.
    • Choose time format and date order without leaving the popup.

Note

This extension does not require an account and does not upload your saved deadlines. Your data stays in chrome.storage.local on the current browser.

Warning

Imported deadlines are a convenience layer, not a replacement for official conference announcements. Submission schedules can change, so always verify important deadlines on the conference website before submitting.

Installation

CCF DDL Tracker is built as a Manifest V3 Chrome extension. You can install it from the Chrome Web Store or load it unpacked from source during development.
